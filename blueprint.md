
# Lotto Number Generator

## Overview

A simple web application to generate and display lottery numbers.

## Implemented Features

*   **UI:**
    *   A main heading "Lotto Number Generator".
    *   A button to trigger the number generation.
    *   A section to display the generated lottery numbers.
*   **Functionality:**
    *   Generates 6 unique random numbers between 1 and 45.
    *   Displays the numbers in a visually appealing way.
